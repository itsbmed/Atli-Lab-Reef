# Reef Pilot

Kundenportal für die ICP-Wasseranalyse von Meerwasseraquarien (ATI · NYOS).

## Struktur

```
reef-pilot/
├── client/   # Frontend — Vue 3 + Vite
└── server/   # Backend — Express (MVC)
```

## Entwicklung

```bash
# Frontend
cd client && npm install && npm run dev

# Backend
cd server && npm install && npm run dev
```
